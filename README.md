# dictionary
自分の人生で知りえた言葉をまとめた辞書
見出し語の説明には自分が認識している意味を記述しているため、正確とは限らない
よくわかっていないものごとについては見出し語を作らない

# ユーザー用ページ
公開したら誰でも見れるページ
ログイン不要
編集権限なし
## 表紙ページ
辞書の表紙的なもの
実際の本における奥付、裏表紙とかの情報や最終更新日時とかも記載する
## リストページ
見出し語のリストを見れるページ
ソート、検索、ページネーション等の機能がある
## 詳細ページ
見出し語の詳細を見れるページ
内容は読み、タグ、詳細
今後、できれば詳細の中に別の見出し語があればリンクになるようにしたい(イメージはニコニコ大百科)

# 管理者ページ
管理者である自分だけが閲覧できるページ
ログイン必須
編集権限あり
CSSやJavaScriptはあまり拘らなくていい
## 新規登録ページ
見出し語を新規登録するページ
見出し語、読み、タグ、詳細の1つ目を登録できる
登録ボタンをクリックすると見出し語編集ページに遷移する
## 見出し語編集ページ
見出し語を編集するページ
ここで直接編集できるのは見出し語、読み、タグだけ
詳細の追加と編集は別途ボタンがある
更新ボタンをクリックすると更新されたこのページに戻ってくる
## 詳細追加ページ
詳細を追加するページ
詳細のIDで開く、挿入する動作を管理している
詳細の文だけ入力できる
追加ボタンをクリックすると見出し語編集ページに戻る
## 詳細編集ページ
詳細を編集するページ
詳細のIDで開く、更新する動作を管理している
詳細の文だけ編集できる
更新ボタンをクリックすると見出し語編集ページに戻る
## タグ追加ページ
新種のタグを追加するページ
このページに既存の見出し語を表示して、所持タグを追加できる構造が理想
## タグ編集ページ
既存のタグの名前を変更するページ
タグの所持状況には影響を与えない
